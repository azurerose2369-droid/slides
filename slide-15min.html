まっさらからやる手順
スキルを持っていない人が最初にやること
1
Claude Code をインストールする
claude.ai/code からダウンロード、またはターミナルで npm install
npm install -g @anthropic-ai/claude-code
5〜10分
2
ワークスペース（フォルダ）を作り CLAUDE.md を置く
「このフォルダで何をするか」を CLAUDE.md に書いておくと、毎回説明しなくていい。基本ルールや自分の役職・目的を書く。
15〜20分
3
slides スキルを入れる（または「作って」と頼む）
スキルファイルを持っている人からもらうか、Claude に「スライドを作れるスキルを作って」と頼む。.claude/skills/ フォルダに置くだけで動く。
一度だけ
4
素材を渡して「スライドにして」と頼む
分析データ・docxファイル・メモ・箇条書き、何でも渡してOK。「5枚にして」「前後半の比較を入れて」など追加指示も自由。
毎回1〜5分
5
ブラウザで確認 → そのまま共有・保存
生成されたHTMLをブラウザで開くだけ。← → キーで操作、キャラコメント付き。GitHub Pages に上げれば URL共有も可能。
1〜2分
